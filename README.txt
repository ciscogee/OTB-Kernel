1. How to Build
    - choose branch you want to build with.
        ex. $ git checkout OTB-OC-TW
        ex. $ git checkout OTB-OC-AOSP
	- cd to Kernel
	    cd Kernel/
	- run build script
	    $ ./build_kernel.sh
	    
2. Update zip
    - Update zip will be in main Kernel/ dir

3. How to Clean
	- run clean script
		$ ./clean
